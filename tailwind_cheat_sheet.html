Tailwind CSS Cheat Sheet

Typography

  • text-lg — Large text
  • font-bold — Bold text
  • text-gray-700 — Gray text color
  • tracking-wide — Letter spacing
  • truncate — Ellipsis overflow

Spacing

  • m-4 — Margin 1rem
  • px-6 — Padding X 1.5rem
  • space-y-4 — Vertical gap between children

Colors

  • bg-blue-500 — Blue background
  • text-white — White text
  • hover:bg-blue-600 — Hover effect
  • bg-gradient-to-r from-green-400 to-blue-500

Flexbox & Layout

  • flex — Make element flex container
  • justify-between — Space between items
  • items-center — Center vertically
  • gap-4 — Gap between children

Interaction

  • cursor-pointer — Pointer cursor
  • transition duration-300 — Smooth animation
  • focus:outline-none — Remove focus ring
  • hover:scale-105 — Scale on hover

Responsive

  • sm:text-sm — Small screen text size
  • md:w-1/2 — Medium screen width 50%
  • lg:flex-row — Flex direction row on large screens
  • xl:hidden — Hidden on extra large screens
